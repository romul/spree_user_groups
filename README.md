SpreeUserGroups
===============

Introduces user groups and calculator to add adjustment for specified user group.


Installation
============

      Add `gem "spree_user_groups", :git => "git://github.com/romul/spree_user_groups.git"`
      Run `bundle install`
      Run `rake db:migrate`
